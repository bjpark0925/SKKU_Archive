실습 1
사용자로부터 입력 받은 수로부터 9단까지 구구단을 출력하는 프로그램을 작성하세요. 단, 0을 입력하면 구구단을 출력하지 않도록 합니다.
▪ 중첩 for 문을 사용해서 작성하세요.
▪ Hint : 중첩 for 문 사용, if else 문 사용, break 문 사용
▪ 입력 : 출력할 구구단의 숫자를 입력
▪ 처리 : 입력 받은 구구단 처리 및 0을 입력 받은 경우에 대한 처리
▪ 출력 : 구구단 출력

실습 2
사용자로부터 입력 받은 수로부터 9단까지 구구단을 출력하는 프로그램을 작성하세요. 단, 0이나 9보다 큰 수를 입력하면 구구단을 출력하지 않고, 프로그램을 종료합니다. 
▪ 중첩 while 문과 무한루프 개념을 사용하세요.
▪ Hint : 중첩 while 문 사용, 무한루프 사용, if 문 사용, break 문 사용
▪ 입력 : 출력할 구구단의 숫자를 입력
▪ 처리 : 입력 받은 구구단 처리 및 0과 9보다 큰 수를 입력 받은 경우에 대한 처리
▪ 출력 : 구구단 출력

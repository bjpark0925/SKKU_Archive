실습 1
주어진 문자열에서 이름 부분을 수정하여 새로운 문구를 생성하는 프로그램을 작성해 보세요. 단, 이름의 슬라이싱 범위는 입력 받아 처리하세요.
▪ Hint : 문자열 포맷 코드 or format() 함수 사용, 문자열 더하기 사용
▪ 입력 : 원본 문자열, 대치할 이름 문자열
▪ 처리 : 원본 문자열에서 이름 부분에 해당하는 부분을 대치할 이름 문자열로 수정
▪ 출력 : 수정된 문자열 출력

실습 2
생일 축하 메시지를 생성하는 프로그램을 작성해 보세요. 생일을 맞이한 사람의 이름, 몇 번째 생일, 보내는 사람의 이름을 입력 받아서 문장을 생성하세요.
▪ Hint : 문자열 포맷 코드 or format() 함수 사용, 이스케이프 코드 사용
▪ 입력 : 생일을 맞이한 사람의 이름, 몇 번째 생일, 보내는 사람의 이름
▪ 처리 : 입력 받을 자료로 생일 축하 노래 및 축하 메시지 작성
▪ 출력 : 생성된 문자열 출력
*이스케이프 코드: \n, \t, \\, \', \"

실습 3
제일 친한 친구 5명의 이름을 리스트에 저장했다가 출력하는 프로그램을 작성하세요.
▪ Hint : append() 함수 사용
▪ 입력 : 5명의 친구 이름
▪ 처리 : 입력 받은 자료를 리스트에 저장
▪ 출력 : 수정된 리스트 출력
반복문 사용 ver, 사용 x ver

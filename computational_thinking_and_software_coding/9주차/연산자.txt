실습 0
동전 교환 프로그램
◼ 입력(input): 교환할 금액
◼ 실행(processing): 입력 받은 금액을 500원, 100원, 50원, 10원짜리로
교환 처리하는 연산 수행
◼ 출력(output): 500원 짜리, 100원 짜리, 50원 짜리, 10원 짜리의 갯수와
동전으로 바꾸지 못한 금액을 출력하여 확인

실습 1
우리 가게에는 3가지 종류의 주스를 판다. 사과 주스는 2000원, 바나나 주스가 3000원, 키워 주스가 4000원이라고 할 때, 각 주스별 판매수에 따른 총 매출을 구하는 프로그램을 작성하세요.
▪ Hint : input() 함수 사용, 산술 연산자 사용
▪ 입력 : 각 주스별 판매량
▪ 처리 : 각 주스별 판매량에 따른 매출 계산 및 총 매출 계산
▪ 출력 : 총 매출 출력

실습 2
자동 판매기에 투입한 돈과 산 물건의 가격을 입력 받아 잔돈을 계산하여 500원짜리 동전과 100원짜리 동전으로 되돌려주는 프로그램을 작성하세요. 물건의 가격은 100원 단위라고 가정합니다.
▪ Hint : input() 함수 사용, 산술 연산자 사용
▪ 입력 : 투입한 돈, 산 물건의 가격
▪ 처리 : 투입한 돈과 산 물건의 가격에 따른 잔돈을 계산하고, 500원
짜리 동전과 100원짜리 동전이 몇 개씩 필요한지 계산
▪ 출력 : 계산한 결과 출력

성적이 90점이상이면 A, 80점 이상은 B, 70점 이상은 C, 60점 이상이면 D, 60점 미만은 F를 주는 성적 산출 프로그램을 작성해보세요.

▪ 입력 : 100점 만점으로 산출한 점수
▪ 처리 : 점수에 맞는 학점 연산
▪ 출력 : 점수에 해당하는 학점 출력

추가 실습 2
입력된 연도가 윤년인지 아닌지를 판단해서 알려주는 프로그램을 작성해 보세요.
▪ Hint : 논리연산자와 조건문 if () ~ else 사용
▪ 입력 : 4자리로 된 연도
▪ 처리 : 입력된 연도가 윤년인지 연산
▪ 윤년 판단하기
1. 4로 나누어 떨어지는 연도
2. 4로 나누어 떨어지지만 100으로 나누어서 떨어지는 연도는 제외
3. 단, 400으로 나누어 떨어지는 연도는 윤년
▪ 출력 : 연산 결과 출력

심화 실습
1~50까지의 수 가운데서
1. 짝수이면서 3의 배수인 수
2. 3의 배수 또는 7의 배수에 해당하는 수
3. 5의 배수가 아닌 수

▪ 3개의 숫자들을 구분하는 프로그램을 작성해 보세요.
▪ Hint : 리스트 사용, 반복문 for() 사용, 조건문 if() 사용
▪ 입력 : 1~50의 수
▪ 처리 : 조건에 맞는 수로 구성된 리스트 생성
▪ 출력 : 생성된 리스트 출력

이상한 작명소제출완료
100점
문제

어떤 작명소에서는 개명 요청이 들어오면 항상 이름의 두 번째 글자와 세 번째 글자를 바꾼 이름을 제시해준다. 또한 이 작명소는 까다로워서, 이름이 세 자 미만이거나 50자를 초과하면 작명을 거부한다. 이때, 기존 이름을 입력받아 개명된 이름과 기존 이름을 출력하는 프로그램을 작성하시오.

객체를 사용해야하고 기존 이름도 기억해두어야하기 때문에 복사생성자를 활용하여 프로그램을 만들어야한다. 구분을 위해 생성자가 실행될 때는 ‘+’를 출력하고 복사생성자가 실행될 때는 ‘#’을 출력한다.

<string> 라이브러리는 사용하지 않는다.





입력

영어로 된 이름이 하나 주어진다. 공백이 포함된 이름은 주어지지 않는다.





출력

생성자와 복사생성자가 실행됨에 따라 +와 #이 한 줄씩 출력된다.

그 후 개명된 이름과 원래 이름이 한 줄씩 출력된다.

허용되지 않는 길이의 이름이 입력되면 invalid를 출력한다.





입/출력 예시
:
공백
:
줄바꿈
:
탭
예시 1
입력
steve
출력
+
#
setve
steve

입력
longnameeelongnameeelongnameeelongnameeelongnameeex
출력
invalid

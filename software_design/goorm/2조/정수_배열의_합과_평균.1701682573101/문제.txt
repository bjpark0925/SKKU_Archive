정수 배열의 합과 평균제출완료
100점
키워드: 포인터, 동적 할당, 배열

문제

 동적으로 할당한 정수 배열의 합과 평균을 계산하는 C++ 프로그램을 작성해야 합니다. 사용자로부터 배열의 크기를 입력 받고, 해당 크기만큼의 정수를 입력 받아 배열을 생성한 후, 합과 평균을 계산하여 출력하는 프로그램을 작성합니다.

조건:
- 배열의 크기와 요소들은 사용자로부터 입력 받는다.
- 동적 할당을 사용하여 배열을 생성한다.
- 합과 평균은 각각 함수를 통해 계산한다.
- 배열과 관련된 메모리는 반드시 해제되어야 한다.

입력

첫 번째 줄에는 배열의 크기를 나타내는 정수가 입력된다.

두 번째 줄에는 배열의 각 요소를 나타내는 정수들이 띄어쓰기로 구분되어 입력된다.

출력

첫 번째 줄에는 배열 요소들의 총합을,

두 번째 줄에는 배열 요소들의 평균을 출력한다. 평균은 소수점 이하 두 자리까지 출력한다.


입/출력 예시
:
공백
:
줄바꿈
:
탭

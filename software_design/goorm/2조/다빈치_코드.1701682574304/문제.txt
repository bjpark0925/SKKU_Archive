다빈치 코드제출완료
100점
키워드: 클래스, 객체 배열, 문자열

문제

 숫자가 적힌 색깔 카드를 배열하려고 한다. 색깔은 흰색과 검은색이 있으며 각각 0부터 11까지의 숫자가 적혀있다.

 왼쪽이 작은 수, 오른쪽이 큰 수가 되도록 배열해야 하며 검은색과 흰색이 같은 숫자일 경우 검은색을 왼쪽에 놓는다.




입력

첫 번째 줄에는 받을 카드의 개수인 정수 T가 주어진다.

두 번째 줄에는 자신이 받게 될 카드의 상태가 주어진다. 색깔과 숫자가 연이어서 주어지며 B는 검은색을, W는 흰색을 뜻하고 뒤에 오는 숫자가 그 카드의 숫자가 된다. (2 ≤ T ≤ 11)

세 번째 줄에는 출력의 방식을 결정하는 정수가 입력된다.



출력

세 번째 줄에 0이 입력된 경우 배열이 끝난 자신의 카드 전부를 출력한다.

만일 특정 숫자를 입력할 경우 해당 순서의 카드 정보를 출력한다. 단, T를 넘는 숫자는 입력되지 않는다. (검은색은 Black, 흰색은 White로 출력한다)



입/출력 예시
:
공백
:
줄바꿈
:
탭
예시 1
입력
3
B5W1B3
0
출력
W1B3B5
예시 2
입력
3
B5W1B3
1
출력
White1
예시 3
입력
3
B5W1B3
3
출력
Black5
⋇ 입출력 형식을 잘 지켜주세요

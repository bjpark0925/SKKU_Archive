단어 비교(동적할당)제출완료
100점
문제

공백이 없는 문자열 두 개를 각각 동적할당 방식을 이용하여 입력받고, 입력 받은 두 문자열이 같은 경우 '동일', 두 문자열이 다른 경우 '다름'을 출력하시오. 

*다만 동적할당과 string을 이용해서 문제를 해결하시오)*

입력

첫째 줄에는 입력할 단어 두 개를 띄어쓰기를 기준으로 작성한다.

출력

두 단어가 같을 경우 '같음'을 다를 경우 '다름'을 출력한다.



입/출력 예시
:
공백
:
줄바꿈
:
탭
예시 1
입력
stdiostdio
출력
같음
예시 2
입력
stringstdlib
출력
다름
⋇ 입출력 형식을 잘 지켜주세요

나만의 벡터(복사생성자)제출완료
100점
문제

‘MyVector’는 정수형 데이터를 저장하는 벡터를 나타내는 클래스이다. ‘MyVector’ 클래스에는 복사 생성자가 필요하다.

*주의사항*

입력 벡터를 복사한 새로운 벡터를 생성하여 출력해야 한다.

복사생성자를 반드시 활용해야 한다.

입력

‘MyVector’ 클래스 객체를 생성할 때 사용하는 입력은 다음과 같다.

- 첫 번째 줄: 정수 ’n’ (1 <= n <= 100) - 벡터의 크기

- 두 번째 줄: 길이가 ’n’인 정수 배열, 공백으로 구분된 정수

출력

입력 벡터를 복사한 벡터를 출력해야 한다.

입/출력 예시
:
공백
:
줄바꿈
:
탭
예시 1
입력
5
12345
출력
OriginalVector:12345
CopiedVector:12345
예시 2
입력
3
345
출력
OriginalVector:345
CopiedVector:345
⋇ 입출력 형식을 잘 지켜주세요

문자열 길이 비교(동적할당)제출완료
100점
문제

문자열 두 개를 각각 동적할당 방식을 이용하여 입력받고, 입력 받은 두 문자열의 길이를 비교하여 길이가 같다면 '동일', 길이가 다르다면 '다름'을 출력하세요. 메모리 소비를 최대한 줄이는 방향으로 코드를 작성해주세요.

입력

첫째 줄과 둘째 줄에 비교하고 싶은 문자열을 입력합니다.

출력

문자열의 길이가 같을 경우 "동일한 길이입니다."를, 다를 경우 "다른 길이입니다."를 출력하게 하시오.

입/출력 예시
:
공백
:
줄바꿈
:
탭
예시 1
입력
Hello
World!
출력
다른길이입니다.
예시 2
입력
Test
Test
출력
동일한길이입니다.
⋇ 입출력 형식을 잘 지켜주세요

개수 세기 문제(디폴트 매개변수)제출완료
100점
문제

함수를 하나만 정의하여 정수 a이상 b이하인 정수의 개수를 구해보자. 두 수 사이의 n의 배수의 개수도 구하고 싶다.

(a ≤ b), n은 1 이상의 정수, 함수는 1개만 이용한다.

*다만 코드 입력창에 제시된 countMultiple 함수를 완성시키고 디폴트 매개변수를 이용하여 문제를 해결하시오*

입력

첫째 줄에는 입력할 인수의 개수 M (2 or 3)을 입력한다. 둘째 줄에는 M=2일 경우 a,b / M=3일 경우 a,b,n을 입력한다.

출력

M=2일 경우 a이상 b이하 정수의 개수를 출력한다. M=3일 경우 a이상 b이하 n의 배수의 개수를 출력한다.



입/출력 예시
:
공백
:
줄바꿈
:
탭
예시 1
입력
2
1321
출력
9
예시 2
입력
3
10507
출력
6
⋇ 입출력 형식을 잘 지켜주세요

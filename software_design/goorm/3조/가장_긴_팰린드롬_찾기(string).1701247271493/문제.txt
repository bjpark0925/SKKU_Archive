가장 긴 팰린드롬 찾기(string)제출완료
100점
문제

앞뒤를 뒤집어도 똑같은 문자열을 팰린드롬(palindrome)이라고 한다. 주어진 문자열에서 가장 긴 팰린드롬 부분 문자열을 찾는 프로그램을 작성해라.

입력

문자열을 무작위로 입력한다. (1 이상 1000 이하의 길이를 가질 수 있음)

출력

주어진 문자열에서 가장 긴 팰린드롬 부분 문자열 출력한다.

(길이가 같은 경우 먼저 나오는 팰린드롬 문자열을 출력해주세요!)

입/출력 예시
:
공백
:
줄바꿈
:
탭
예시 1
입력
babad
출력
bab
예시 2
입력
abbcccdddd
출력
dddd
⋇ 입출력 형식을 잘 지켜주세요

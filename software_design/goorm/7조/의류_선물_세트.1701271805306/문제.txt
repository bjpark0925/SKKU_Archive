의류 선물 세트제출완료
100점
문제

온라인 의류 쇼핑몰에서 추첨 행사를 진행하여 당첨자에게 3개의 의류 선물세트를 증정하려고 한다. 다음의 과정을 수행하시오.

선물세트1 생성: 선물 세트의 이름, 의류의 개수, 각 의류의 종류, 색상, 가격을 입력받아 생성한다.
선물세트2 생성: 첫 번째 선물 세트를 복사한 후 선물 세트의 이름을 새롭게 입력받아 변경하고 대체될 의류 인덱스, 대체할 의류의 종류, 색상, 가격을 입력받아 변경한다.
선물세트3 생성: 선물 세트의 이름을 새롭게 입력받고 첫 번째, 두 번째 선물세트를 합쳐서 커플을 위한 2인 선물세트를 만든다.
3개의 선물세트의 상세정보를 출력한다.( 아래 [출력 예시]의 형식을 따라야 한다.)
[조건]

PresentSet 클래스 멤버 함수를 완성하여 문제를 해결한다.
Clothes 클래스와 ParentSet 내부의 멤버 변수는 수정할 수 없다.
첫 번째 선물세트에 옷을 추가할 때 add_clothes() 함수를 사용한다.
두 번째 선물세트의 이름을 바꿀 때 set_name() 함수를 사용한다.
두 번째 선물세트를 생성할 때 replace() 함수를 사용한다.
선물세트를 복사할 때는 복사 생성자를 이용한다.
커플을 위한 2인 선물 세트를 만들 때는 첫 번째 선물 세트 객체와 두 번째 선물 세트 객체를 인수로 전달받는 참조 매개변수를 가진 생성자를 이용하여야 한다.


입력

첫 번째 줄에 선물세트1의 이름이 주어진다. 

두 번째 줄에 선물세트1에 포함된 의류의 개수 이 주어진다. (  )

세 번째 줄부터  번 줄까지 선물세트1에 포함된 의류의 종류, 색상, 가격이 공백으로 구분되어 주어진다. 

 번째 줄에 선물세트2의 이름이 주어진다.

 번째 줄에 대체될 의류의 인덱스 번호  가 주어진다. (  )

 번째 줄에 대체할 의류의 종류, 색상, 가격이 공백으로 구분되어 주어진다. 

마지막 줄에 선물세트3의 이름이 주어진다. 

(단, 선물세트의 이름과 의류의 종류와 색상은  공백을 포함하지 않는 길이 10 이하의 문자열이고, 가격은 100만 이하의 양의 정수이다.)



[입력 예시]

Aset			// 첫 번째 선물 세트 이름 
3			// 첫 번째 선물 세트의 의류 개수 
coat brown 180000	// 첫 번째 선물 세트의 첫 번째 의류 
T-shirt blue 60000	// 첫 번째 선물 세트의 두 번째 의류 
beggyjeans gray 70000	// 첫 번째 선물 세트의 세 번째 의류 
Bset			// 두 번째 선물 세트 이름 
1			// 대체될 의류 인덱스 
mustang black 230000	// 대체할 의류 정보 
Cset		        // 커플 세트의 이름


출력

각 선물세트에 대한 정보를 개행 하나를 두고 구분하여 출력한다. 각 선물 세트에 대해 다음과 같이 출력한다.

첫 줄에 선물세트의 이름을 출력한다.
그 다음 줄부터 번호와 함께 선물 세트에 포함된 의류의 종류, 색상, 가격을 한 줄에 하나씩 출력한다.
의류의 종류, 색상, 가격은 다음과 같은 양식으로 출력한다:
-   {number}. {종류}({색상}): {가격} WON
자세한 사항은 [출력 예시]를 참고한다.

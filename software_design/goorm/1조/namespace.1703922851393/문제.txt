namespace제출완료
100점
문제
해당 소스코드가 작동하도록 하시오.

단, 다음 조건을 따라야 한다.

    1) main 함수를 변경할 수 없다.

    2) "using"를 사용할 수 없다.

           2-1) "using namespace" 금지

           2-1) "using std::cout", "using std::cin" 등도 금지

    3) main함수 이외에서 "std::cin", "std::cout"을 사용할 수 없다.

입력
입력은 ' '로 구분되어 있는 영단어 두 개이다.

(입력 문자열은 1000자 미만이다.)

ex) hello hi

출력
입력한 문자열이 출력된다.

입/출력 예시
:
공백
:
줄바꿈
:
탭
예시 1
입력
hellohi
출력
hello
hi
예시 2
입력
myname
출력
my
name
예시 3
입력
ab
출력
a
b
예시 4
입력
id2019000000
출력
id
2019000000
⋇ 입출력 형식을 잘 지켜주세요

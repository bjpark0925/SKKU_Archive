음식 주문 Class 작성 문제제출완료
100점
﻿문제
아래의 조건을 바탕으로 각각의 Class를 구현하세요.

- 음식 클래스 (Food):

Food 클래스는 음식의 이름(name), 가격(price), 주문 수량(quantity)을 저장합니다.
모든 멤버 변수는 private 으로 선언되어야 합니다.
기본 생성자를 통해 이름을 "Unknown", 가격을 0.0, 수량을 0 으로 초기화합니다.
음식의 이름과 가격을 받는 생성자를 작성하세요.

- 주문 클래스 (Order):

Order 클래스는 여러 음식 객체를 저장하고 관리합니다.
Order 클래스는 음식 객체의 동적 배열을 사용하여 음식들을 저장합니다.
addFood() 함수를 통해 음식을 주문에 추가할 수 있습니다.
calculateTotalPrice() 함수를 통해 현재 주문의 전체 가격을 계산합니다.
displayOrderInfo() 함수를 통해 현재 주문의 상세 정보를 출력합니다.

- 고객 클래스 (Customer):

Customer 클래스는 주문 정보를 가지고 있습니다.
기본 생성자를 통해 고객의 이름(name)을 "Unknown"으로 초기화합니다.
고객의 이름을 받는 생성자를 작성하세요.
placeOrder() 함수를 통해 주문을 추가하고, displayCustomerInfo() 함수를 통해 고객의 주문 정보를 출력합니다.
입력
별도의 입력은 없습니다. 다음 main 함수에 적혀있는 부분이 그 역할을 대신합니다.





출력
손님의 주문 정보를 출력합니다. 출력 형식은 "이름, 주문 음식 1 : 가격 1, 주문 음식 2 : 가격 2, ..., Total: 총 가격" 입니다.

입/출력 예시
:
공백
:
줄바꿈
:
탭
예시 1
출력
CustomerInformation:
JohnDoe,CheeseBurger:$8.99FrenchFries:$3.99
Total:$12.98
⋇ 입출력 형식을 잘 지켜주세요
